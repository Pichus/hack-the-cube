Що таке сайт для браузера?

В основному це просто набір файликів з розширенням .html, .css та .js

Про html

Hypertext Markup Language. У html файликах розташовується розмітка файлу, його "кістки"

Код

index.html

Результат

Про css

Cascading Style Sheets. У css файликах описується зовнішній вигляд сайту. Наприклад, можна змініити колір елементу, його розташування на екрані, розмір та багато іншого

Код

styles.css

Результат

Hello, world!

Про js

JavaScript. На відміну від html та css, js - це мова програмування, це вам не іграшки. Завдяки js можна додавати інтерактивні елементи на свій сайт, зробити так, щоб користувач почувався так, наче він користується справжнім застосунком, а не просто сайтом.
